---------------------------
OPENING ETERNAL CATERPILLAR
---------------------------

To start Eternal Caterpillar on Windows, unzip "EternalCaterpillar.zip" and open "EternalCaterpillar.exe". Note that all files inside the new directory must live in that directory or the game won't work.

On Mac, open the "EternalCaterpillar.app" file.

---------------------------
PLAYING ETERNAL CATERPILLAR
---------------------------

Use the arrow keys to select the options, and press space or enter to begin playing. From there, use the arrow keys to move the caterpillar around. The game is over when the caterpillar hits the wall or itself.
